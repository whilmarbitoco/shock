Template Engine

Variables, conditionals with else/elseif, loops with empty-state handling — compiled to AST nodes and rendered server-side.

Template syntax
<h1>{{title}}</h1>

{% if user != null %}
    <p>Hello, {{user.name}}</p>
{% elseif guest %}
    <p>Hello, Guest</p>
{% else %}
    <p>Please log in</p>
{% endif %}

{% for item in items %}
    <li>{{item}}</li>
{% empty %}
    <li>No items found</li>
{% endfor %}
Controller usage
res.view("dashboard", Map.of(
    "user", currentUser,
    "items", itemList,
    "guest", false
));
Lexer to AST to Render
Lexer lexer = new Lexer(template);
Parser parser = new Parser(lexer);
List<Node> nodes = parser.parse();

for (Node n : nodes)
    output.append(n.render(ctx));

Active Record ORM

Models extend Repository<T>. Define relations with @HasMany and @BelongsTo. Migrations run automatically or generate from annotations.

Model definition
@Table("users")
public class User {

    @Column
    private int id;

    @Column
    private String name;

    @Column
    private String email;

    @HasMany(
        foreignKey = "user_id",
        related = Post.class
    )
    private List<Post> posts;
}
Querying
UserRepository users =
    new UserRepository();

// Find by primary key
User u = users.find(1);

// Find all
List<User> all = users.all();

// Eager load relations
User u = users.with("posts")
              .find(1);

// Create
User u = new User();
u.setName("Whilmar");
u.setEmail("w@dev.com");
users.save(u);

Middleware Pipeline

Global middleware runs on every request. Named middleware can be attached to specific routes. Auth, logging, sessions, CORS — plug them in where needed.

Global middleware
public void global() {
    addGlobal(new LogsMiddleware());
    addGlobal(new SessionMiddleware());
    addGlobal(new CorsMiddleware());
}
Per-route middleware
public void register() {
    add("auth", new AuthMiddleware());

    // Apply to specific routes
    router.get("/dashboard",
        DashboardController.class,
        "index",
        "auth");  // requires auth
}
