Shock

A modern, lightweight, and efficient Java MVC web framework for building robust and scalable web applications.

Sample App

Key Features

  • Modern Architecture: Built with a focus on modern design principles and best practices.
  • Lightweight & Fast: Minimal overhead and optimized for high performance.
  • MVC Pattern: Clean and organized code with a strong separation of concerns.
  • Dynamic Routing: Flexible and intuitive routing system for easy URL mapping.
  • Customizable Error Handling: Create elegant and informative error pages.
  • Scalable: Designed to handle growing traffic and complexity.
