A lightweight MVC framework built from scratch

Routing, ORM, migrations, template engine, middleware — no Spring, no servlet container, no magic. Just a ServerSocket and clean Java.

Get Started Live Demo
Application.java 
public class Application {

    public static void main(String[] args) {
        RouteRegistry routes = new Routes();
        MiddlewareRegistry mw = new Middlewares();

        Application app = new Application(routes, mw);
        app.run();
    }
}

Core features

No servlet container

Runs on a raw ServerSocket. No Tomcat, no Jetty, no Spring Boot. Starts in milliseconds.

Active Record ORM

HasMany, BelongsTo relations, eager loading, migrations — all built in. No Hibernate needed.

Dynamic routing

Clean route definitions with middleware support. GET, POST, PUT, DELETE, PATCH — all first-class.

Middleware pipeline

Global and per-route middleware. Auth, logging, sessions, CORS — plug them in where you need them.

Template engine

{{variable}}, {% if %}, {% for %}, {% else %} — simple, fast, no external deps.

Multi-database

MySQL, PostgreSQL, SQLite via dialect abstraction. Maven profiles keep your POM clean.

Quick start

1 Clone 
git clone https://github.com/whilmarbitoco/shock.git
cd shock
2 Configure 
# config.properties
db.url=jdbc:sqlite:data.db
server.port=8080
3 Run 
mvn package -P sqlite
java -jar target/shock.jar
